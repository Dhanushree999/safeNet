# safeNet
SafeNet - A cybersecurity tool that detects unsafe websites, explains risks, and tracks user activity.
SafeNet is a beginner-friendly cybersecurity tool that detects unsafe websites.

## Features
- Detects unsafe websites
- Explains why a site is unsafe
- Tracks history of checked websites
- Counts dangerous attempts

## Built With
- Python
- Tkinter (GUI)
